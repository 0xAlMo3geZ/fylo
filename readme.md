# Fylo Website

Fylo landing page with dark/light mode switcher for tailwindcss practice.

[Frontend Mentor Challenge](https://www.frontendmentor.io/challenges/loopstudios-landing-page-N88J5Onjw)

## Usage

Install dependencies

```
npm Install
```

Run Tailwind CLI

```
npm run watch
```
